# JerseyHolic 新系统功能清单总表

> 版本：v2.0 | 更新日期：2026-04-16
> 来源系统：OpenCart 3.0（A站买家前台）、ThinkPHP 6.1（B站支付后台）、新增功能

## 优先级说明（MoSCoW 方法）

| 优先级 | 含义 | 说明 |
|--------|------|------|
| **P0** | Must Have | 上线必备，系统核心功能 |
| **P1** | Should Have | 尽快完成，缺失严重影响业务 |
| **P2** | Could Have | 资源允许时实现 |
| **P3** | Won't Have | 本期不做，记录备忘 |

## 复杂度说明

| 标记 | 含义 | 预估工时 |
|------|------|---------|
| S | 简单 | 1-2 天 |
| M | 中等 | 3-5 天 |
| L | 复杂 | 1-2 周 |
| XL | 极复杂 | 2-4 周 |
